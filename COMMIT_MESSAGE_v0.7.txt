## System Improvements Referenced in Docs

### PCG System
- Room-based procedural generation with level_id and room_code
- Door system with E-key proximity interaction
- Area visualization overlay (spawn zones, combat areas, safe zones)
- Runtime configuration via config/pcg_config.json
- Seed-based reproducible generation

### Debug Tools
- F9: Wall jump debug with state visualization
- F10: Area overlay with adjustable opacity
- F5: Comprehensive debugger menu with grid navigation
- F8: Enhanced tile inspector with metadata display
- Camera zoom system (Z key)

### Physics & Movement
- Physics-based wall jump system with coyote time
- Wall slide with controlled descent
- Double-tap Space for no-clip + floating mode
- Enhanced air control and momentum preservation

### UI/UX
- Centralized input handling via InputHandler class
- E-key door interaction with proximity prompts
- Camera zoom with visual feedback
- Enhanced HUD with active cheat indicators

### Architecture
- Component-based entity system (CombatComponent, PhysicsComponent, VisionComponent)
- Tile system with TileRegistry, TileType enums, and collision detection
- Modular debug overlay system
- Preserved legacy level system for compatibility
